MCP-Powered Skill Management 基于 MCP 的技能管理系统

Mix Skills, 混合技能，
Empower AI 智能赋能

Skillix is an MCP-based skill management system that provides AI coding agents with the ability to create, manage, load, and evolve skills. Skillix 是一个基于 MCP 的技能管理系统，为 AI 编码代理提供技能的创建、管理、加载和进化能力。

Get Started 快速开始 GitHub
~/.cursor/mcp.json
// Add Skillix to your MCP configuration // 将 Skillix 添加到你的 MCP 配置
{
  "mcpServers": {
    "skillix": {
      "command": "npx",
      "args": ["skillix-mcp"]
    }
  }
}

Features 功能特性

Everything you need for AI skills AI 技能管理的一站式解决方案

A comprehensive toolkit for managing, evolving, and sharing AI skills. 一套完整的工具包，用于管理、进化和共享 AI 技能。

Skill Management 技能管理

Create, read, update, and delete local skills with built-in version control and automatic backup. 创建、读取、更新和删除本地技能，内置版本控制和自动备份。

Smart Triage 智能分流

Intelligent task analysis that automatically recommends the best skill for your task. 智能任务分析，自动推荐最适合的技能。

Skill Market 技能市场

Search, install, and share skills from remote sources with the community. 从远程源搜索、安装技能，与社区分享。

Local-First 本地优先

Project skills take precedence over global skills for context-aware resolution. 项目技能优先于全局技能，实现上下文感知。

Version Rollback 版本回退

Built-in backup and version history. Never lose your skill improvements. 内置备份和版本历史，永不丢失改进。

Configuration 配置管理

Flexible global and project-level configuration with multiple skill sources. 灵活的全局和项目级配置，支持多个技能源。
