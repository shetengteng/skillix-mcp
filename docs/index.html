MCP-Powered Skill Management 基于 MCP 的技能管理系统

Mix Skills, 混合技能，
Empower AI 智能赋能

Skillix is an MCP-based skill management system that provides AI coding agents with the ability to create, manage, load, and evolve skills. Skillix 是一个基于 MCP 的技能管理系统，为 AI 编码代理提供技能的创建、管理、加载和进化能力。

Get Started 快速开始 GitHub
~/.cursor/mcp.json
// Step 1: Add Skillix to your MCP configuration // 第一步：将 Skillix 添加到你的 MCP 配置
{
  "mcpServers": {
    "skillix": {
      "command": "npx",
      "args": ["skillix-mcp"]
    }
  }
}

// Step 2: Ask Cursor AI to initialize Skillix // 第二步：让 Cursor AI 初始化 Skillix
// "Please initialize Skillix for this project" // "请为这个项目初始化 Skillix"

Quick Start 快速开始

Get started in 3 steps 三步开始使用

Simple setup process to enable Skillix in your project. 简单的设置流程，在项目中启用 Skillix。

Setup Guide 设置指南
1

Install MCP Server 安装 MCP Server

Click the button below to auto-install, or manually add to ~/.cursor/mcp.json 点击下方按钮一键安装，或手动添加到 ~/.cursor/mcp.json

One-Click Install in Cursor 一键安装到 Cursor
2

Initialize Project 初始化项目

Ask Cursor AI to run init command, it will setup .skillix/ and install Cursor Rule 让 Cursor AI 执行 init 命令，会自动设置 .skillix/ 并安装 Cursor Rule

"Please initialize Skillix for this project" "请为这个项目初始化 Skillix"
3

Start Using 开始使用

The installed Cursor Rule will automatically trigger sx-dispatch for task analysis 安装的 Cursor Rule 会自动触发 sx-dispatch 进行任务分析

.cursor/rules/skillix.mdc ✓
